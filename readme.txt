Adds a widget to display images and optionally a caption

== Description ==

Adds a simple widget that allows the display of an image with an optional link and a caption.

== Installation ==


1. Upload `quicktake-image-widget` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. After installing the plugin you should see a new widget in the widget administration area.
